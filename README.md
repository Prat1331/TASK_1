## Installation

1. Clone the repository:

    ```bash
   https://github.com/Prat1331/TASK_1.git
    ```

2. Navigate to the project directory:

    ```bash
    cd TASK_1-Project
    ```

3. Install the required dependencies:

    ```bash
    pip install -r requirements.txt
    ```
